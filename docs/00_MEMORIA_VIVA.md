# StayFlow - Memoria Viva Del Proyecto

Ultima actualizacion: 2026-05-14

## Objetivo

Esta carpeta contiene los documentos vivos de StayFlow. No son documentos finales cerrados: deben cambiar junto con el producto.

## Regla general

Cada avance importante debe revisar si impacta alguno de estos documentos:

- Documento Comercial.
- Manual De Uso.
- Checklist Final Del Producto.
- Resumen Legal Y Fiscal Inicial.
- Que Deberia Decir La Web Legalmente.
- Ecosistema / Ideas Futuras.

Si una decision cambia el producto, tambien debe cambiar la documentacion correspondiente.

## Documentos

- `01_DOCUMENTO_COMERCIAL.md`: venta, posicionamiento, modulos y discurso.
- `02_MANUAL_DE_USO.md`: uso paso a paso para personas no tecnicas.
- `03_CHECKLIST_FINAL_DEL_PRODUCTO.md`: control de calidad antes de mostrar o vender.
- `04_RESUMEN_LEGAL_Y_FISCAL_INICIAL.md`: mapa inicial legal/fiscal.
- `05_QUE_DEBERIA_DECIR_LA_WEB_LEGALMENTE.md`: textos y paginas legales de la web.
- `06_ECOSISTEMA_E_IDEAS_FUTURAS.md`: limites del producto actual e ideas futuras.
- `07_GUION_VIDEO_DEMO.md`: guion base para grabar video explicativo.
